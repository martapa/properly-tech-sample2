# Properly Technical Sample

## Getting started

### Stack
- Node
- Express
- MongoDB
- React

#### Basic commands

1. Start server 
- `yarn start`
- `yarn logs`

2. Start client 
- `cd client`
- `yarn start`

3. Run tests for server
- `yarn test:watch`

4. Run tests for React
- `cd client`
- `yarn test`
